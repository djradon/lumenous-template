---
title: Url
description: Field to store url values
---

The field of type `url` is used to store URLs. It display a `<input type="url">`
in the UI.

## Example

```ts
{
  name: "href",
  type: "url",
}
```

## Available options

This field only has
[the common options](../configuration/fields.md#common-field-options).
