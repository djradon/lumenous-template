---
title: Date
description: Field to store date values
---

The field of type `date` is used to store date values. It display a
`<input type="date">` in the UI.

## Example

```ts
{
  name: "published",
  type: "date",
}
```

## Available options

This field only has
[the common options](../configuration/fields.md#common-field-options).
