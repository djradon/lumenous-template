---
title: Number
description: Field to store numeric values
---

The field of type `number` is used to store URLs. It display a
`<input type="number">` in the UI.

## Example

```ts
{
  name: "position",
  type: "number",
}
```

## Available options

This field only has
[the common options](../configuration/fields.md#common-field-options).
