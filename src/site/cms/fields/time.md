---
title: Time
description: Field to store time values
---

The field of type `time` is used to store time values. It display a
`<input type="time">` in the UI.

## Example

```ts
{
  name: "starts_at",
  type: "time",
}
```

## Available options

This field only has
[the common options](../configuration/fields.md#common-field-options).
