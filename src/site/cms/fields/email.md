---
title: Email
description: Field to store email values
---

The field of type `email` is used to store emails. It display a
`<input type="email">` in the UI.

## Example

```ts
{
  name: "contact",
  type: "email",
}
```

## Available options

This field only has
[the common options](../configuration/fields.md#common-field-options).
