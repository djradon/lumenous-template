Code

Field to store code content

  1. Example
  2. Available options

The field of type code is used to store code of any type, like HTML, CSS, JavaScript, etc. It display a codemirror editor in the UI.

Example

{
  name: "content",
  type: "code",
}

Available options

This field only has the common options.
