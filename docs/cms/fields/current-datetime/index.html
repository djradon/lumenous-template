Current datetime

Field to store the current date time

  1. Example
  2. Available options

The field of type current-datetime is very similar to datetime with the difference of the value is updated automatically every time the document is saved. It's useful to store the last modification time of a post or article.

Example

{
  name: "last_modified",
  type: "current-datetime",
}

By default, the field is user-editable. If you want to make it read only:

{
  name: "last_modified",
  type: "current-datetime",
  attributes: {
    readonly: true
  }
}

Available options

This field only has the common options.
